Popular tickets on LightHouse!

This web application uses the LightHouse REST API to show the popular tickets (ie with the most watchers).

Samples

Playframework 2.x

@controllers.routes.Application.popularTickets("play", "82401")

View popular tickets

Playframework 1.x

@controllers.routes.Application.popularTickets("play", "57987")

View popular tickets

CakePHP

@controllers.routes.Application.popularTickets("cakephp", "42648")

View popular tickets

How to use

You can use this tool to check the popular tickets of your favorite framework, if the bugtracking is provided by LightHouse.

To do this, you just have to build the proper Url inspired by the Url of your LightHouse project.

Here is an example of a LightHouse Url:

https://play.lighthouseapp.com/projects/57987-play-framework/overview

The first part (in green) is called the "account name", and the second one (in blue) is the "project id".

To view the popular tickets, just use:

/play/projects/82401/tickets/popular
